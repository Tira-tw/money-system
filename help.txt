1.修改config.json的資料 :
owner_id
prefix
token
max
min
new
    
owner_id填你的id
    
prefix填你的前綴
    
token就填token
    
max每次簽到最高獲得數
    
min每次簽到最低獲得數
    
new創建帳號初始金額

2. pip install discord

3. python bot.py

4. done!



小提示 : 
開發者可以使用[prefix]give <ID> <多少錢>
