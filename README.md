# 金錢系統
> 金錢系統是一個非常方便的機器人 , 擁有7個功能![有一個隱藏功能] <br>
# 機器人邀請
> 目前還不公開Bot~ <br>
# 設定
> 1.修改config.json的資料 :
`owner_id` <br>
`prefix` <br>
`token`<br> 
`max`<br>
`min`<br>
`new`<br><br>
    
owner_id填你的id <br> <br>
    
prefix填你的前綴 <br> <br>
    
token就填token <br> <br>
    
max每次簽到最高獲得數 <br> <br>
    
min每次簽到最低獲得數 <br> <br>
    
new創建帳號初始金額 <br> <br>

> 2. pip install discord <br> <br>

> 3. python bot.py <br> <br>

> 4. done! <br> <br>

# 小提示 
開發者可以使用[prefix]give <ID> <多少錢> <br> <br>
